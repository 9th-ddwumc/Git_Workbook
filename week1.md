# 1주차 (Git)

## Git이란
- 버전 관리 목적: 변경 이력 추적, 협업
- 작업 영역: Working Dir -> Staging Area -> Local Repo -> Remote Repo
- .gitignore: 추적 제외 파일/ 폴더 관리

## 명령어
- 상태: git status
- 스테이징: git add . / git add <file>
- 커밋: git commit -m "커밋 메시지"
- 푸시: git push origin <브랜치>

## 브랜치
- 생성: git branch <브랜치>
- 이동: git switch <브랜치>
- 병합: git merge <브랜치>
